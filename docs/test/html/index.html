LivermoreBigArtificialNeuralNetworkToolkit
Overview

This is the API documentation for LBANN ( Livermore Big Artificial Neural Network). LBANN is a toolkit for deep learning training on HPC systems. To get started with LBANN please refer to the Getting Started pages for installation and basic usage information. To see detailed information on the neural network features available in LBANN refer to the following pages:
