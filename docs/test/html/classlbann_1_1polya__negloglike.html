Compute the Polya negative log-likelihood objective function. Given the parameters of the predicted Polya distribution, the ground truth label counts , and total counts the Polya negative log-likelihood is

This function updates the objective function value with the mean Polya negative log-likelihood across the mini-batch. Note that each column of the predictions and ground truth matrices should have non-negative entries.
